# ⚡ PROJECT ELECTRIFY: THE WORKBENCH

A diegetic electrical engineering simulator.

## 🚀 HOW TO PLAY
1. Pick a level from the **Pegboard**.
2. Drag components from the tray to the grid.
3. Press **'R'** to rotate components.
4. **Right-click** to remove components.
5. Click **ENGAGE** to run the simulation.

## 🛠️ TECH STACK
- **Frontend:** React 18 + Tailwind CSS + Motion.
- **State:** Zustand.
- **Simulation:** Custom iterative Nodal Analysis solver.
